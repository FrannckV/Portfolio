ToDo

  • Device: iPhone, iPad
  • Framework: SwiftUI
  • Project URL: https://github.com/FrannckV/ToDo

Task manager iOS application with Core Data integration. The main features let us add tasks, list and delete them. This project supports the light/dark appearance and has a set of configurations that allows the user to change the theme color and the app icon as well.

  1. Todo Application with Core Data and SwiftUI
  2. Saves and retrieves data from the local database
  3. Deletes data and refreshes the database
  4. Displays piece of data dynamically in a List View
  5. Shows an image and a quote when there is no todo item in the List
  6. Advanced Settings View
  7. External website link with SwiftUI
  8. iOS Alternate App Icons with SwiftUI
  9. App Themes with SwiftUI
