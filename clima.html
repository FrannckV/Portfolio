Clima

  • Device: iPhone
  • Framework: Storyboard
  • Project URL: https://github.com/FrannckV/Clima

Beautiful, dark-mode enabled weather app. This project checks to grab live data from an API that gives the weather for the current location based on the GPS data from the iPhone as well as by searching for a city manually.

  1. Dark-mode enabled app with Storyboard
  2. Usage of the UITextField to get user input
  3. Delegate pattern
  4. Swift protocols and extensions
  5. Swift computed properties
  6. Swift closures and completion handlers
  7. URLSession to network and HTTP requests
  8. Parses JSON with the native Encodable and Decodable protocols
  9. Usage of the Core Location to get the current location from the phone GPS
