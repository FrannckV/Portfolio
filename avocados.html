Avocados

  • Device: iPhone, iPad
  • Framework: SwiftUI
  • Project URL: https://github.com/FrannckV/Avocados

Eye-catchy layout in SwiftUI. The Avocados app has a fairly complex UI in SwiftUI developed with custom View Modifiers, avoiding code repetition. This project supports light and dark appearance and a set of images showed in different colors. This app is optimized for the iPad as well. It supports the wider screens both in landscape and portrait.

  1. Scroll Views inside another Scroll View
  2. Adaptive Colors and Images
  3. Custom View Modifiers
  4. Tab View with custom icons
  5. Supports the Light / Dark modes with SwiftUI
