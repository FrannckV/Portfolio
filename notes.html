Notes

  • Device: Apple Watch
  • Framework: SwiftUI
  • Project URL: https://github.com/FrannckV/Notes

Modern Notes application for Apple watches. Standalone watch extension using the watchOS and SwiftUI frameworks together.

  1. Permanently saves notes and store data on Apple Watch
  2. Shows a list of notes on Apple Watch
  3. Selects and deletes notes from Apple Watch
  4. Shows SwiftUI views conditionally
  5. Uniform design language across pages
  6. Swift's Codable protocol for serialization
  7. Custom user interface elements
  8. Input tools: Dictate, Scribble, Emoji, Type
