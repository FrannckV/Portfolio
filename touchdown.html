Touchdown

  • Device: iPhone
  • Framework: SwiftUI
  • Project URL: https://github.com/FrannckV/Touchdown

Modern E-commerce mobile application with the SwiftUI framework. iOS 14 app prototype that shows a basic home and product screen with animations and different stacks as well as a personalized set of colors and JSON decoded information.

  1. @Environment property wrapper usage
  2. Observable Object property wrapper
  3. Data model shared across the whole app
  4. Custom navigation bar
  5. Custom shape with Shape Path
  6. Horizontal grid layouts in SwiftUI
  7. JSON files decoded and fetched data with SwiftUI
