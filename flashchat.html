FlashChat

  • Device: iPhone
  • Framework: Storyboard
  • Project URL: https://github.com/FrannckV/FlashChat

Flash Chat is an internet based messaging app similar to WhatsApp, the popular messaging app that was bought by Facebook for $22 billion. This project will be using a service called Firebase Firestore as a backend database to store and retrieve the messages from the cloud.

  1. Integrates third party libraries in, using Cocoapods and Swift Package Manager
  2. Stores data in the cloud using Firebase Firestore
  3. Queries and sorts the Firebase database
  4. Uses Firebase for user authentication, registration and login
  5. Works with UITableViews and sets the data sources and delegates
  6. Customs views using .xib files to modify native design components
  7. Embeds View Controllers in a Navigation Controller and applies the navigation stack
  8. Usage of constants file and static properties to store Strings and other constants
  9. Usage of the App Lifecycle by applying viewWillAppear and viewWillDisappear
  10. Creation of direct Segues for navigation
